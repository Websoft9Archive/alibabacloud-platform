# Windows Connect

You can connect to the Window server through the Remote Desktop Tool (MSTSC) on your local computer, or you can use a Powershell connection.

Below is the steps of MSTSC connection

  1. Log in AWS Portal, get you Public IP Address of VM

  2. Choose a way to open a local computer remote desktop (three-in-one):

    • Open Start -> Remote Desktop
    • Open Start, input "mstsc" directly, the system will search for the Remote Desktop
    • Using the keyboard Windows Logo + R to start the command windows, input input "mstsc" to open the Remote Desktop

  1. In the Remote Desktop Connection dialog box, enter the public IP address of the instance. Click Show Options.

  2. Enter the user name,check Allow me to save credentials. In this way, you do not need to manually enter the password again when you log on later.

  3. Click the Connect button to complete connection to the instance.

  4. After logging in remotely, you can copy the file directly from the local and paste the file to the server.

  5. If you need to use FTP, you need to install the FTP software yourself (Recommended to use FileZilla Server)
